# ⚡ JPDCL Bill Calculator

A modern and accurate electricity bill estimator for JPDCL domestic consumers.

## 🌐 Live Website

https://arunbhagatt.github.io/jpdcl-bill-calculator/

---

## ✨ Features

- Modern glassmorphism UI
- Accurate slab-wise bill calculation
- Demand charge calculation
- Mobile-friendly design
- Installable as Android app (PWA)
- Fast and lightweight
- Works directly in browser

---

## 📊 Tariff Logic Used

### Energy Charges

| Units | Rate |
|-------|------|
| 0 – 200 | ₹2.30/unit |
| 201 – 400 | ₹4.00/unit |
| Above 400 | ₹4.30/unit |

### Demand Charges

```text
Sanctioned Load × ₹8
```

Default sanctioned load:
```text
2 kW
```
